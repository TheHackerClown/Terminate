## Terminate

A 2d-Shooter Game based on nothings, its just a shooter game with custom infinite world generation 
algorithm, which is hosted on [Map_Generation_Algorithm](https://github.com/TheHackerClown/2D-Mapgen).

The Backend Server used for the multiplayer section of this game is hosted on [Backend_Server](https://github.com/TheHackerClown/Terminate-Backend/).
